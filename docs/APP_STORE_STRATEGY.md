## 4. Privacy & Compliance

### Privacy Policy Requirements

Must disclose:
- Audio data processing (not stored)
- Translation text (optional storage for history)
- Analytics data collection
- Third-party services used

### Data Collection Declaration

**iOS App Privacy Labels**
- Data Used to Track You: None
- Data Linked to You: Email (if registered), Usage Data
- Data Not Linked to You: Diagnostics

**Google Play Data Safety**
- Data shared: None
- Data collected: App activity, App info

### GDPR Compliance
- Consent management for EU users
- Data export functionality
- Right to deletion
- Clear privacy notices

### COPPA Compliance
- Age gate if targeting children
